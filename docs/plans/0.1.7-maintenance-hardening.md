# PawnLogic 0.1.7 Maintenance Hardening Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Prepare a focused 0.1.7 maintenance release that prevents documentation drift, continues low-risk `run_turn()` and streaming cleanup, and strengthens release safety checks without changing user-facing agent behavior.

**Architecture:** Keep all public runtime contracts stable: chat messages keep the same shape, provider stream deltas remain dict-based, workspace cleanup commands keep the same output fields, and provider/model behavior does not change. Improve the system by adding guard tests around documentation naming, extracting more internal helpers from large functions, broadening typed-island coverage, and automating release consistency checks.

**Tech Stack:** Python 3.10+, pytest, Ruff, mypy typed island, GitHub Actions, setuptools dynamic versioning, PyPI Trusted Publishing.

---

## Release Theme

0.1.7 should be a maintenance release. It should make future releases harder to
ship with stale docs, make core loops easier to modify, and add targeted
regression coverage around safety-sensitive behavior.

## Non-Goals

- Do not add new model providers, model aliases, tools, or commands.
- Do not change visible tool-call JSON, assistant message shape, or
  `reasoning_content` persistence.
- Do not introduce a public stream event class or dataclass protocol.
- Do not replace `AgentSession.run_turn()` with LangGraph or a new runtime
  dependency.
- Do not bundle third-party skill packs into PyPI distributions.
- Do not rename default English docs; `README.md` and `GUIDE.md` remain the
  canonical English files.

## Current Baseline

- 0.1.6 is published to PyPI and merged to `main`.
- Chinese documentation filenames use the formal locale suffix:
  `README_zh-CN.md` and `GUIDE_zh-CN.md`.
- `tools/check_doc_structure.py` compares English docs against the zh-CN docs.
- `AgentSession.run_turn()` has helper extraction for preparation,
  empty-response recovery, assistant/tool-call append, plan guard, concurrency
  limit, and tool-batch execution.
- `core/api_client.py` has provider payload/header helpers and an internal
  `_read_sse_lines()` selector.
- CI runs Ruff, typed-island mypy, Python 3.11 fast tests, release matrix tests,
  dynamic E2E, and docs checks.

```python
"""Validate release-facing documentation agrees with config.paths.VERSION."""

from __future__ import annotations

from pathlib import Path
import re


VERSION_RE = re.compile(r'^VERSION = "([0-9]+\.[0-9]+\.[0-9]+)"$', re.MULTILINE)


def _read(path: Path) -> str:
    return path.read_text(encoding="utf-8")


def _version(root: Path) -> str:
    match = VERSION_RE.search(_read(root / "config" / "paths.py"))
    if match is None:
        raise ValueError("config/paths.py must contain VERSION = \"x.y.z\"")
    return match.group(1)


def check_repository(root: Path) -> list[str]:
    version = _version(root)
    errors: list[str] = []

    expectations = {
        "README.md": f"The current public release is **{version}**.",
        "README_zh-CN.md": (
            f"\u5f53\u524d\u516c\u5f00\u53d1\u5e03\u7248\u672c\u662f **{version}**\u3002"
        ),
        "CHANGELOG.md": f"## [{version}] - ",
        "SECURITY.md": f"| {version}   | ✅ Yes",
    }
    forbidden = {
        "README.md": "The current public release is **0.1.6**.",
        "README_zh-CN.md": (
            "\u5f53\u524d\u516c\u5f00\u53d1\u5e03\u7248\u672c\u662f **0.1.6**\u3002"
        ),
    }

    for relative_path, expected in expectations.items():
        text = _read(root / relative_path)
        if expected not in text:
            errors.append(f"{relative_path} is missing expected release text: {expected}")

    for relative_path, stale in forbidden.items():
        text = _read(root / relative_path)
        if version != "0.1.6" and stale in text:
            errors.append(f"{relative_path} still contains stale 0.1.6 public release text")

    return errors


def main() -> int:
    errors = check_repository(Path(__file__).resolve().parents[1])
    if errors:
        for error in errors:
            print(f"- {error}")
        return 1
    print("Release consistency check passed.")
    return 0


if __name__ == "__main__":
    raise SystemExit(main())
```
